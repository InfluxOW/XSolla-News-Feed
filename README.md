## Requirements
PHP ^7.2.5
PHP extensions: mbstring, curl, dom, xml, zip, sqlite

## Commands
1. Run `make setup` to setup the project.
2. Run `make test` to run tests.
3. Run `make seed` to seed the database with fake data.
